# Installation Instructions

```
eg:SG8A_AGON_G2Y_B1_AGX_Orin_YUV_JP6.2_L4TR36.4.3

1.After extracting, enter the source folder.

2.Place hardware/nvidia/t23x/nv-public/overlay/SG8A_AGON_G2Y_B1 into the corresponding directory in the local SDK.

3.Add the corresponding declarations to the Makefile in the parent directory.

4.Place source/nvidia-oot/drivers and source/nvidia-oot/include into the corresponding directories in the local SDK.

5.Add the corresponding declarations to the Makefile in the parent directory.

6.Specify the corresponding device tree files and driver files in the compilation environment.

7.Compile.
```
